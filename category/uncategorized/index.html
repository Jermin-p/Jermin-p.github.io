My first WordPress website

October 16, 2025October 16, 2025 bruh1 Comment

I decided to create my own website so I can post my blogs, projects, work experience, and my achievements. I believe this will help me share my insights about anything to anyone, to get known by other people, and serve as a reference for myself to see where I am now.

I created this website using WordPress. I initially used it on my local machine first by installing the software Local to run WordPress. This is new to me since I’ve only built websites using HTML and CSS. I am not a pro at the moment in terms of building that. On the other hand, I was trying out or experimenting with WordPress for a few hours until I got the hang of it. I was constantly changing themes and designs because I could not find the right design. I burnt most of my hours designing then just picking another theme or doing the website design from scratch since I didn’t like it, or sometimes I found it very hard to design because there are different website builders I think and some themes generally don’t work like the others. But I found this Coral White theme and I decided to use it because it’s simple and easy to use.

To host this website, my first instinct was to use hosting platforms like Hostinger, GoDaddy, etc. But I do not want to pay since this is my first website that I will try to keep running for as long as I can. And I have a history of leaving AWS platform machines running with inefficient configurations that led me to have a high bill without noticing it. And maybe my site could get attacked by DDoS that would eventually give me high bills. I could do that, make my site secure and efficient in terms of running it, but it would take me a lot of time to learn that, so I decided to move on from that idea.

After some research, I learned that GitHub offers free website page hosting. So I decided to go with it. I’ve converted this WordPress to static then pushed it to my repository. Now that this website is running, I’ll do my best to post contents/blogs and documentation for my projects.
